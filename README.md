# Générateur de Personnages RPG Étendu

Application Java de gestion de personnages RPG permettant de créer, personnaliser et organiser des héros dans des équipes et armées hiérarchisées. Le projet implémente 10 design patterns (Builder, Decorator, DAO, Composite, Singleton, Command, Chain of Responsibility, MVC) à travers un système de jeu complet.
